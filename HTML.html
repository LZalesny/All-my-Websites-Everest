HTML is a langue that people use to code websites. HTML is important for us becasue this language makes up all the sites on the internet. HTML stands for hypertext markup language. It began in 1989 and created by Tim Berners-Lee in Geneva Switzerland. CSS is important becasue it gives a website the control of the look and feel of content written in HTML. CSS stands for Casading Style Sheets. Its used to sytle a website. The idea of CSS began in 1994 by Hakon Wium Lie while working with Tim Berners-Lee. It was finally released in December 17, 1996. The way tags work in HTML is puting a less than sign < then a greater then sign > A tag works by implamenting an attribute inbetween the signs. Examples of attributes are background color="#FF3300", title="HTML", a href="url", size attribute width and height. The color attribute lets you to change the color of the text. Title attribute gives you the title of the webpage. The href attribute lets you be able to insert a link into your webpage. The size attribute allows you to change the height and width of an image that is to large or to small.
